The script is generated dynamically based on your configuration selections and includes your Data Layer. The Data Layer is generated from the data sources listed in the Data Sources tab.

To place the code on your website:

  1. Ensure you are are getting the code for the correct environment by looking at the navigation on the left. (Dev, QA, Prod)
  2. Click on the "Tealium Script" tab above.
  3. Click and drag to select all the code.
  4. Copy the code.
  5. Navigate to the appropriate webpage, and paste the code after the opening BODY tag.
